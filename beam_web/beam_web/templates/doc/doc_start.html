Zielsetzung

Es soll eine Verbindung zwischen einem Beamer und einem beliebigen Endgerät hergestellt werden, um den Bildschirminhalt des Endgeräts über den Beamer auszugeben. Dazu ist es notwendig, eine VNC Verbindung zwischen dem Endgerät und den zum Beamer gehörenden Accesspoint / Linux Minirechner herzustellen. Der prinzipielle Funktionsaufbau ist nachzulesen unter VNC.


Betriebsmodus

Es gibt zweierlei Betriebsmodi:

Zum einen den unmoderierten Modus, der aktiv ist solange kein Dozent/Moderator eingeloggt ist.
Hier können Nutzer des Accesspoints streng in der Reihenfolge der Anmeldung ihre eigene VNC Verbindung herstellen.
Dies wird über eine FiFo Liste gewährleistet.

Zum anderen den Dozentenmodus, der aktiv wird, sobald ein Dozent/Moderator sich eingeloggt hat.
Er übernimmt die Steuerung, und kann manuell selbst die Liste verwalten, die zur Übertragung Bereitschaft gemeldet haben.
